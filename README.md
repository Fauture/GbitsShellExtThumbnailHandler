# GbitsThumbnailHandler
问道Gbits缩略图程序

 ## 注意
最新版需要卸载原始扩展才能使用

 ## 使用
需要管理员权限执行

注册
``` C:\Windows\SysWOW64\Regsvr32.exe %~dp0GbitsShellExtThumbnailHandler.dll ```

卸载
``` C:\Windows\SysWOW64\Regsvr32.exe /u %~dp0GbitsShellExtThumbnailHandler.dll ```

 ## 说明
测试环境Windows 10 64 1909/21H1。
